## lab1
Моделирование базовых случаийных величин

## tasks
Осуществить моделирование n = 1000 реализаций БСВ с помощью мультипликативного конгруэнтного метода (МКМ) с параметрами a0, b0, M = 2^31.
Проверить точность моделирования с помощью критерия согласия Колмогорова и chi-2-критерия Пирсона с уровнем значимости alpha = 0.05.
